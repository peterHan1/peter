1、退款金额怎么计算出来的？

答： 如果您的订单中使用了优惠工具（优惠券、参加满减活动等），退款金额会根据订单中每件商品的价格按比例分配，交易的形式以及优惠情况不同，则计算方式略有不同，以实际退款的金额为准。

如： 您的订单中A商品价格100元，B商品价格200元，订单使用了30元优惠券，实付270元。如果整笔订单都退货，则退还270元，使用优惠券返回；

则退货A时，A商品会退还83.34元（退款商品的退款金额＝退款商品提交订单时的达人价 —退款商品提交订单时的达人价商品总金额*优惠券金额）。注： 具体退还金额以实际核定金额为准！

2、退款或售后成功，为什么没有收到退款？

答： 钱款会原路退回到您下单的支付宝、微信或银行账号中，财务受理需要3-5个工作日，请您关注。

3、退款成功后，如何查询钱款去向？

答： 如果您的订单显示退货成功，钱款会原路径退回至您的账号中，请关注您的支付方式和到账时间。

如果您选择在线支付（支付宝、微信），将会在财务受理后3-5个工作日原路退款至您支付的平台。例如：您使用支付宝绑定的银行卡支付的金额会退回到银行卡账号中。

如您使用银行卡/信用卡（招商银行）支付，将会在财务受理后3-5个工作日内退还至您下单支付时使用的银行卡/信用卡账户。
